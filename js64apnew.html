Event

Events are specific actions which are performed at a specific place.

Event Handling

Event Handling is a process of specific actions to be performed when an event occurs.

Event Bubbling

body --> onLoad--> p,h1,table

Life Cycle of an Event

  • The user performs an action to raise an event
  • The event object is updated to determine the event state.
  • The event is fired.
  • The event bubbling occurs as the event bubbles through the elements of the heirarchy.
  • The event handler is invoked that performs the specified actions.
